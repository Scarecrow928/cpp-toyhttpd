toyhttpd demo

简介

服务器结构仿照tinyhttpd，改用C++实现，并加入了多线程处理每个请求。
相比tinyhttpd，支持的更多文件类型，并且可以通过增加MIME类型字典加入更多类型：
  • text: html, txt, css, javascript, json等等
  • image: png, jpg, gif等等

编译

直接编译运行：
  • 安装g++，make
  • make
  • export PORT=端口号
  • ./toyhttpd
用docker：
  • docker build -t mytag .
  • docker run -p 12000:12000 -v $(pwd)/htdocs:/http/htdocs -v $(pwd)/default:/http/default mytag
