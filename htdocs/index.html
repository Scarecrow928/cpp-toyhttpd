toyhttpd demo

简介

服务器结构仿照tinyhttpd，改用C++实现，并加入了多线程处理每个请求。
相比toyhttpd，支持的更多文件类型，并且可以通过增加MIME类型字典加入更多类型：
  • text: html, txt, css, javascript
  • image: png, jpg, gif

demo

  • jpg
  • gif
  • 较大文件，以text/plain传输
  • txt
  • 404

编译

直接编译运行：
  • 安装g++，make
  • make
  • export PORT=端口号
  • ./toyhttpd
有docker：
  • docker build -t mytag .
  • docker run -p 12000:12000 -v $(pwd)/hrdocs:/http/htdocs -v $(pwd)/default:/http/default mytag
